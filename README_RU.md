# Другая Вселенная

Русская версия расширения для SillyTavern.

Оригинальный проект: https://github.com/mangkoodkung/another-universe

## Изменения

- Весь пользовательский интерфейс переведён на русский.
- Выбор языка пользовательского интерфейса удалён.
- Язык адаптированной анкеты: как в оригинале, русский или английский.
- Добавлено окно адаптации анкеты с выбором размера и собственным промптом.
- Добавлена универсальная механика «Яйцекладка».
- Исправлены правила промптов, чтобы `{{char}}` не смешивался с данными `{{user}}`.
- Greeting и first_mes не должны писать действия или реплики за `{{user}}`.
